# Claude Code Global Config Templates

Platform-specific `CLAUDE.md` templates for `~/.claude/CLAUDE.md`:

- **[CLAUDE-windows.md](CLAUDE-windows.md)** — Windows 11 + fnm + pnpm + PowerShell
- **[CLAUDE-macos.md](CLAUDE-macos.md)** — macOS + nvm + Homebrew + zsh

Copy the appropriate file to `~/.claude/CLAUDE.md` on each machine, then customize as your tools change.
